# Installation and User Manual

# AI-Based Student Performance Prediction and Classification System

## System Requirements

- Python 3.11 or above
- Node.js 18 or above
- npm
- Git
- VS Code (Recommended)

---

# Installation Steps

## Step 1: Clone the Repository

```bash
git clone https://github.com/Adnan0505/student-performance-prediction-system.git
cd student-performance-prediction-system
```

---

## Step 2: Backend Setup

Open Terminal

```bash
cd project/backend
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment (Windows)

```bash
venv\Scripts\activate
```

Install Required Packages

```bash
pip install -r requirements.txt
```

Run Backend Server

```bash
python app.py
```

Backend URL:

```
http://localhost:5000
```

---

## Step 3: Frontend Setup

Open another terminal

```bash
cd project/frontend
```

Install Dependencies

```bash
npm install
```

Run Frontend

```bash
npm run dev
```

Frontend URL:

```
http://localhost:5173
```

---

# How to Use the Application

1. Open the frontend in your browser.
2. Enter the required student details.
3. Click the **Predict** button.
4. View the predicted student performance generated by the AI model.

# Troubleshooting

### Backend is not starting

- Check that Python is installed.
- Activate the virtual environment.
- Install dependencies using:

```bash
pip install -r requirements.txt
```

### Frontend is not starting

Run:

```bash
npm install
npm run dev
```
